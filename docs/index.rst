STM32MP135F eBook Reader Documentation
======================================

Welcome to the documentation for ``STM32MP135F eBook Reader``.

This project started out of curiosity. I own an e-book reader that I use every day, and I wanted to understand in detail how such a device works. A second motivation was to learn embedded Linux product development and use this project as a demonstrator for my consulting work.

This documentation contain information about how one can start development of the device and documentation for various code APIs i developed within the project.

I hope you find the code and docs clear, high quality, and informative, this project is my way of showcasing what I can build and how I work.
